Project Title:

Customer Shopping Behavior Analysis

Project Overview:
This project analyzes customer shopping behavior to identify purchasing patterns, spending trends, and key customer segments. The goal is to provide actionable insights that can support better business decisions such as targeted marketing, inventory planning, and customer retention strategies.

Objectives:
Analyze customer purchasing patterns
Identify high-value customers
Understand product and category performance
Visualize key metrics for easy decision-making

Tools & Technologies:
Power BI – Data visualization and dashboard creation
SQL – Data querying and transformation
Excel / CSV – Data preprocessing
Data Modeling – Relationships and calculated measures

Key Insights:
Identified top-spending customers and frequent buyers
Discovered trends in product categories and purchase frequency
Analyzed average order value and customer retention patterns
Highlighted seasonal or behavioral trends in shopping activity

Dashboard Preview:

The Power BI dashboard includes:
KPIs (Total Sales, Average Order Value, Customer Count)
Customer segmentation visuals
Category-wise and time-based sales analysis
(Open the .pbix file in Power BI Desktop to view the dashboard.)
